# Cable Scene Interior Tower Locations and Observation Ranges

## Overview

This document provides exact locations and observation ranges for interior power transmission towers in the Europe Cable Scene. Interior towers are positioned between transmission line endpoints and are critical for maintaining continuous power flow across cable spans.

## Scene Description

The Europe Cable Scene is a 7km x 7km European countryside environment featuring:
- 25 interior power transmission towers (excluding 16 endpoint towers)
- Varied terrain (elevation 258-388m for interior towers)
- Multiple vegetation regions (grass, wheat, corn, desert)
- Road networks and transmission corridors
- Realistic power infrastructure layout

## Interior Tower Location Data

### Purpose
These coordinates provide:
- Exact interior tower positions for mission planning
- Recommended observation offset ranges for cable span monitoring
- Elevation data for terrain-following operations
- Strategic surveillance positioning for continuous transmission lines

### Coordinate Specifications
- **Tower Positions**: Interior towers only (exclude line endpoints)
- **Observation Offsets**: Recommended 15-25 meter horizontal offset ranges
- **Altitude**: Use tower elevation (no altitude offset recommended)
- **Coverage**: Critical transmission line support points

## Observation Guidelines

### Offset Positioning
- **Horizontal Range**: 15-25 meters from tower center
- **Altitude**: Use tower elevation (Z coordinate)
- **Safety Buffer**: Maintain minimum 15m clearance from structures
- **Coverage Pattern**: Position for optimal cable and tower visibility

### Mission Planning
- **Primary Corridors**: Groups 1d, 1e provide highest elevation coverage
- **Dense Networks**: Groups 1h, 1d offer multiple tower configurations  
- **Terrain Variation**: Elevation range 255-402m across all towers
- **Geographic Spread**: Full 7km x 7km scene coverage

### Surveillance Applications
- Power line condition monitoring
- Vegetation encroachment detection
- Tower structural inspection
- Cable sag measurement
- Thermal analysis of electrical components

## Technical Specifications

### Coordinate System
- **Reference Frame**: Scene coordinate system
- **Units**: Meters
- **Origin**: Scene center at (0, 0, 0)
