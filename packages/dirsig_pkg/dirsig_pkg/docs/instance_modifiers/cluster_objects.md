# Cluster Objects Node

**Description:**
Places a group of objects near a specific location in the scene, useful for simulating clusters or collections of similar objects.

## Inputs
- **Objects**: Objects to place.
- **Scene Location (m, m)**: Location in scene coordinates to place the objects, e.g., [0, 0].
- **Number of Objects**: How many objects to place.
- **Radius (m)**: Radius around the scene location to distribute the objects.

## Outputs
- **Objects**: Objects placed in a cluster at the specified location.
