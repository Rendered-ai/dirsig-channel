# Scale Objects Node

**Description:**
Scales the size of one or more objects by specified multiplicative factors along the X, Y, and Z axes.

## Inputs
- **Objects**: One or more objects to scale.
- **Scale Factors**: Multiplicative factors to stretch or shrink the object, e.g., [1.0, 1.0, 1.0].

## Outputs
- **Objects**: Scaled input objects.
