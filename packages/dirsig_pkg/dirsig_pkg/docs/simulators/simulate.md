# Simulate Node (DIRSIG5)

**Description:**
The Simulate node runs DIRSIG5 simulation tasks, generating synthetic imagery, truth, and metadata based on the provided scene, platform, and simulation parameters.

## Inputs
- **Scene**: DIRSIG Scene to be simulated.
- **Sensor**: Platform with Sensor, Motion, and Tasks configuration.
- **Reference Datetime**: The starting datetime for the simulation (ISO 8601 format or a linked value).
- **Start Task Time (s)**: The start time of the task in seconds (default: 0.0).
- **Capture Duration (s)**: Duration of the simulated flight/capture in seconds.
- **Ephemeris**: Ephemeris plugin to set celestial positions (e.g., Sun/Moon angles).
- **Save Radiance**: Whether to save radiance data (True/False).
- **Save ENVI Truth**: Whether to save ENVI truth data (True/False).

## Outputs
- **Imagery**: Synthetic images generated by the simulation.
- **Truth/Metadata**: Associated ground truth and metadata for each frame.
