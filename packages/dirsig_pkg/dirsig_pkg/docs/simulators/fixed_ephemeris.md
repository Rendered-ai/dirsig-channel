# Fixed Ephemeris Node

- **Description**: The Fixed Ephemeris node allows users to directly supply static positions for the Sun and Moon, providing a simple way to set celestial positions for simulations.
- **Inputs**:
  - **Solar Zenith**: The angle of the Sun from the vertical, in degrees.
  - **Solar Azimuth**: The compass direction of the Sun, in degrees.
  - **Lunar Zenith**: The angle of the Moon from the vertical, in degrees.
  - **Lunar Azimuth**: The compass direction of the Moon, in degrees.
  - **Lunar Fraction**: The fraction of the Moon that is illuminated.
- **Outputs**:
  - **Ephemeris**: Provides the ephemeris data for the simulation.
